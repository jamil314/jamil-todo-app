# ToDo App
[Live link](https://jamil-todo-app.vercel.app) <br>
This is a ToDo List manager app written in react<br>
Key technologies used in this project: 
 - React
 - Styled Component
 - Zustand

## To run locally
```
git clone https://github.com/jamil314/jamil-todo-app
cd jamil-todo-app
npm i
npm start
```
